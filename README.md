## üì¶ Getting Started

### Prerequisites
- **Node.js** 16+ or **npm** 8+
- Basic knowledge of React and modern JavaScript

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Anil-sai0418/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   ```
   http://localhost:5173
   ```

### Available Scripts

```bash
npm run dev      # Start development server with HMR
npm run build    # Build for production
npm run preview  # Preview production build locally
npm run lint     # Run ESLint checks
```

## üé® Customization

### Colors & Theme
Edit `tailwind.config.js` to customize:
```javascript
theme: {
  extend: {
    colors: {
      primary: '#your-color',
    }
  }
}
```

### Animations
Modify animation speeds in component files or `tailwind.config.js`:
```javascript
keyframes: {
  moveBackground: {
    '0%': { backgroundPosition: '0 0' },
    '100%': { backgroundPosition: '40px 40px' },
  },
}
```

### Projects
Update project data in `src/components/Project/Projects.jsx`:
```javascript
const PROJECTS = [
  {
    id: 1,
    title: "Your Project",
    description: "Description here...",
    image: "/your-image.png",
    tags: ["Tech1", "Tech2"],
    link: "https://yourproject.com",
    github: "https://github.com/yourrepo"
  }
]
```
