## KEY FUNCTIONS

useScroll():
- Tracks window scroll position
- Returns scrollY MotionValue
- Used for scroll-linked animations

useTransform():
- Creates derived MotionValue
- Maps scroll progress to CSS properties
- Creates parallax, fade, blur effects

useMotionValueEvent():
- Listens to MotionValue changes
- Triggers state updates
- Updates navbar visibility

scrollIntoView():
- Smooth scroll to section
- Called on nav item click
- Auto-closes drawer on mobile
